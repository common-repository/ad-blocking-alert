== Description ==

Stop losing money on your google adsense and other ad network accounts by using Ad Blocking Alert!
It can detect if your visitor is using ad blocking plugins such as AdBlock, and if so will popup a message asking the visitor to disable or add an exception before they can visit the site.

You can completely customize the popup message or show an image.

== Installation ==

How to install Ad Blocking Alert and get it working.

1. Install the plugin
2. Activate the plugin through the 'Plugins' menu in WordPress
3. On your left sidebar click 'Ad Blocking Alert' and set up your page
